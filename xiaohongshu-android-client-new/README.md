# 红薯糖水 - 小红书采集助手

## 功能说明

### 悬浮按钮（4个功能）
| 按钮 | 功能 | 说明 |
|------|------|------|
| 🟢 采集 | 自动采集笔记 | 自动搜索关键词，采集笔记短链接并上传 |
| 🔵 转换 | 长链接转短链接 | 将剪贴板的小红书长链接转为短链接 |
| 🟠 面板 | 显示/隐藏面板 | 切换消息面板的显示状态 |
| 🔴 关闭 | 关闭脚本 | 关闭悬浮窗和消息面板 |

### 消息面板
- 实时显示运行状态
- 显示采集统计（已采集、已上传、缓存）
- 显示运行日志

## 安装步骤

### 1. 下载 AutoX
在手机上下载并安装 AutoX：
- GitHub: https://github.com/kkevsekk1/AutoX/releases
- 下载最新的 APK 文件安装

### 2. 开启权限
打开 AutoX，需要授予以下权限：
- ✅ **无障碍服务**：必须，用于自动化操作
- ✅ **悬浮窗权限**：必须，用于显示悬浮按钮
- ✅ **存储权限**：可选，用于保存日志

### 3. 安装小红书
确保手机已安装小红书 APP 并登录

### 4. 导入脚本
**方式一：通过文件**
1. 将 `main.js` 传到手机
2. 打开 AutoX → 点击 "+" → 选择文件

**方式二：复制粘贴**
1. 复制 `main.js` 的代码
2. 打开 AutoX → 点击 "+" → 新建脚本 → 粘贴代码

### 5. 运行脚本
点击 AutoX 中的 ▶️ 运行按钮

## 使用方法

### 采集笔记
1. 点击悬浮窗的 **采集** 按钮
2. 脚本自动打开小红书并搜索
3. 自动采集笔记短链接
4. 达到缓存数量后自动上传

### 转换链接
1. 复制小红书笔记的长链接
2. 点击悬浮窗的 **转换** 按钮
3. 自动将长链接转为短链接
4. 结果自动复制到剪贴板

### 查看日志
- 确保面板可见（点击 **面板** 按钮）
- 在消息面板中查看实时日志

### 关闭脚本
- 点击 **关闭** 按钮
- 或在 AutoX 中停止脚本

## 常见问题

### Q: 点击没反应？
A: 检查无障碍服务是否开启

### Q: 悬浮窗不显示？
A: 检查悬浮窗权限是否授予

### Q: 采集失败？
A:
1. 确保小红书已登录
2. 检查网络连接
3. 查看消息面板的错误信息

### Q: 上传失败？
A:
1. 检查服务器地址是否正确
2. 确认网络连接正常
3. 联系管理员检查服务状态

## 技术支持

- 使用 AutoX 6.0.0 或更高版本
- Android 8.0 或更高版本
- 需要保持小红书登录状态
